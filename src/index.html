Sever Membrane

Sever Membrane aims to create an Elixir like API to spawn and manage processes in JavaScript

Spawn a process

// spawn(fn -> 1 + 2 end)
spawn(() => 1 + 2)

Get process id

// pid = spawn(fn -> 1 + 2 end)
const id = spawn(() => 1 + 2)

Check if process is alive

// alive?(id)
alive(id)

Use process id

// spawn(fn -> self() == 1 end)
spawn(() => me() === 1)

Send message to service

// pid = spawn(fn -> receive do msg -> IO.puts "Received: #{msg}" end end)
const id = spawn(() => receive((msg) => console.log(`Received: ${msg}`)))

send(id, 'Hello')
// Received: Hello
